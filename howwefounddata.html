Where and how did we find this data?

We found our data from three different places: the first was netindex.com, the second the hub 2010 document which is available at data.gov.uk and the third was from the land value registry which is also available data.gov.uk. To sort through our data we used a combination of human and machine power. For example we sorted through some data by hand but others we had to use python programs to sort through. We then geocoded our data using google fusion datas and finally correlated the data sets together.
